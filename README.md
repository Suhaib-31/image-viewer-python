# Image Viewer Application 🖼️

## Overview

Image Viewer is a desktop application developed using Python and Tkinter. The application allows users to browse and view image files through a graphical user interface.

---

## Features

- Open Images
- JPG Support
- PNG Support
- JPEG Support
- GIF Support
- User-Friendly Interface

---

## Technologies Used

- Python 3
- Tkinter
- Pillow (PIL)

---

## Installation

```bash
pip install pillow
```

---

## Run

```bash
python main.py
```
